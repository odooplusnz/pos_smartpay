Odoo Smartpay POS Integration

Smartpay EFTPOS terminals integration with Odoo's POS Module.

Versions supported: v14 Enterprise +

Overview

This module enables seamless integration between Odoo POS and Smartpay EFTPOS terminals.
In scope:
  • Pairing POS shop with the eftpos terminal. Supports 1 to 1 shop pairings
  • Purchase handling
  • Refund handling
All other operations such as QR codes scanning, tipping, etc. are not supported.

Features

  1. Only the selected fields will be exposed via the API per each model.

  2. Expiry date on access tokens.

  3. Extended and abbreviated searches

    ?detailed=True

  4. Domain search support
    5. ?domain=name,=,S00007
  5. Support for limits in queries
    6. ?limit=2
  6. Support for multiple databases
  7. Supports file uploads
  8. Supports:
    • GET
    • POST
    • PUT
    • DELETE
