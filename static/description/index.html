Odoo Smartpay POS Integration

Smartpay EFTPOS terminals integration with Odoo's POS Module.

Versions supported: v14 Enterprise +

Overview

This module enables seamless integration between Odoo POS and Smartpay EFTPOS terminals.
In scope:
  • Pairing POS shop with the eftpos terminal. Supports 1 to 1 shop pairings
  • Purchase handling
  • Refund handling
All other operations such as QR codes scanning, tipping, etc. are not supported but can be added upon request.

Certified by Smartpay

Features

  • Simplified one-click setup
  • Lightweight
  • Error handling for power failures / Internet faults

Documentation

Full documentation is available via README.md in the root folder, or on our github.
This includes server config, Odoo setup, pairing process and general usage.
