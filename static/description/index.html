Odoo REST API

Enables per model access to your Odoo instance via the REST API.

Versions supported: v14 Enterprise

Overview

This module adds REST API access to your Odoo instance enabling and simplifying integrations with other software packages. It does so via three simple steps: activate the API access on res.user, find the desired model and select which field you would like to have access to.

Each model can be configured individually. Only the required fields are exposed via the API. Supports multiple databases.

Features

  1. Only the selected fields will be exposed via the API per each model.

  2. Expiry date on access tokens.

  3. Extended and abbreviated searches

    ?detailed=True

  4. Domain search support
    5. ?domain=name,=,S00007
  5. Support for limits in queries
    6. ?limit=2
  6. Support for multiple databases
  7. Supports file uploads
  8. Supports:
    • GET
    • POST
    • PUT
    • DELETE
